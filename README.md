# Trader Performance vs Market Sentiment Analysis

##  Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed) and trader behavior on Hyperliquid. The objective is to uncover patterns and derive actionable trading strategies.


##  Methodology
- Data cleaning and preprocessing
- Handling missing values and duplicates
- Converting timestamps and aligning datasets at daily level
- Feature engineering (PnL, win rate, trade frequency, trade size)
- Merging datasets for analysis


##  Key Insights
- Traders show higher risk-taking behavior during Greed periods
- Fear periods exhibit higher volatility and inconsistent performance
- Trade activity increases during strong sentiment phases
- Significant variation exists across trader segments


##  Strategy Recommendations
- Reduce leverage and position size during Fear periods
- Avoid overtrading during Greed periods
- Apply stricter risk management for high-frequency traders
- Use segmentation-based strategies for better performance
